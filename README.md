## 🧠 Features

* 🔎 **Website Vulnerability Scanner**
* 🚪 **Admin Panel Detection**
* 🌐 **Open Port Scanner**
* ⚠️ **Real-time Risk Score Engine**
* 🧬 **AI Attack Prediction System**
* ⏱ **Exploit Time Estimation**
* 🛡 **AI-Based Security Recommendations**
* 📊 **Live Scan Visualization (Charts)**
* 📄 **Downloadable Security Report**

---

## ⚙️ Tech Stack

* **Frontend:** HTML, CSS (Glassmorphism UI), JavaScript
* **Backend:** Python (Flask)
* **AI Engine:** Custom Risk & Attack Prediction Logic
* **Visualization:** Chart.js
* **Concurrency:** ThreadPoolExecutor
* **Security Modules:** Custom Scanners + Nessus-style parsing

## 🚀 How to Run Locally

```bash
# Clone repository
git clone https://github.com/manu-2006/AI-Digital-Twin-Hacker.git

# Go to project folder
cd AI-Digital-Twin-Hacker

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

👉 Open in browser:

```
http://127.0.0.1:5000
```

---

## 📊 How It Works

1. Enter a website URL
2. System scans:

   * Open ports
   * Vulnerabilities
   * Admin panels
3. AI engine:

   * Calculates risk score
   * Predicts attack vectors
   * Estimates exploit time
4. Generates:

   * Visual dashboard
   * Security report
